CIPHER




Es una aplicación web en donde los usuarios, pueden cifrar palabras y descifrar claves, para ello, primero tiene que indicar un desplazamiento específico de caracteres con un número (_offset_). Con el fin que el usuario pueda guardar sus claves en esta misma aplicación de forma segura.
